# Bullet lists

Bulletlists can be created with asterisk, plus and minus

* \*
* \+
* \-

Markdown syntax  
This code segment will create two lists, with a noticeably space between them.
``` 
* list one
* list one
* list one
+ list two
+ list two
+ list two
```
###### Example
* list one
* list one
* list one
+ list two
+ list two
+ list two

Markdown syntax  
This code segment will create a indented list.
``` 
* list
  * indent one
    * indent two
      * indent tree does not work
```
###### Example
* list
  * indent one
    * indent two
      * indent tree does not work
