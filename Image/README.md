Markdown syntax
```

```

## Short image link 2

Reference

Markdown syntax
```
![alt text][logo]

[logo]: https://github.com/adam-p/markdown-here/raw/master/src/common/images/icon48.png "MouseOver Hover Title Text"
```
Example  


![alt text][logo]

[logo]: /Image/User-Story-Card.png "MouseOver Hover Title Text"
